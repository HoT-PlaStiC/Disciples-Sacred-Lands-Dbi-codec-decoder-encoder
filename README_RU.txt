DBI codec for Disciples Sacred Lands
************************************* ——————- made by HoT_PlaStiC
——————-

Этот скрипт позволяет:

    1. вывести список изображений из любого файла .DBI
    2. декодировать изображения из любого файла .DBI
    3. закодировать изображения в любой файл .DBI
    4. заменить изображения в любом файле .DBI

Сначала откройте PowerShell (введите “powershell” в текущей папке и
нажмите Enter).

Для работы необходимо установить библиотеку Pillow.

Введите:

python -m pip install pillow

и нажмите Enter.

    1. Вывести список изображений:
            python fd_portrait_codec_v2_36.py list any.DBI

    2. Декодировать изображения из любого файла .DBI:
            python fd_portrait_codec_v2_36.py decode any.DBI portrék\

        Декодировать все изображения из всех файлов .DBI в текущей папке:
            python fd_portrait_codec_v2_36.py decodeall

    (Вы можете изменить имя папки (portrék\), а также заменить any.DBI
    на any_mod.DBI, если хотите декодировать ранее закодированные изображения.)

    3. Закодировать изображения:

        Для UNIT.DBI:
            python fd_portrait_codec_v2_36.py encode CFDW0048.png UNIT.DBI

    (CFDW0048.png приведён только в качестве примера.)

    (При кодировании в UNIT.DBI изображения должны иметь размер
    55x67 или 115x67 пикселей и быть в формате PNG.)

    (При кодировании нового изображения в UNIT.DBI скрипт автоматически
    создаст новый слот и новый код (начиная с GP001S00),
    поэтому в этом случае имя в конце команды указывать не нужно.)

    Чтобы перезаписать исходный файл .DBI, добавьте флаг "ow":

            python fd_portrait_codec_v2_36.py encode CFDW0048.png UNIT.DBI ow

        Для других файлов .DBI:
            python fd_portrait_codec_v2_36.py encode CFDW0048.png any.DBI FD050S00

    (CFDW0048.png приведён только в качестве примера.)

    (Необходимо указать имя изображения. FD050S00 — лишь пример.)

    Обратите внимание: игра использует систему слоёв,
    поэтому имя определяет, к какой категории слоя относится изображение.
    Используйте правильное имя.

    Чтобы перезаписать исходный файл .DBI, добавьте флаг "ow":

            python fd_portrait_codec_v2_36.py encode CFDW0048.png any.DBI FD050S00 ow

    4. Заменить изображения:
            python fd_portrait_codec_v2_36.py replace CFDW0048.png any.DBI FD026S00

    (CFDW0048.png приведён только в качестве примера.)

    (При работе с UNIT.DBI изображения должны иметь размер
    55x67 или 115x67 пикселей и быть в формате PNG.)

    (Вы можете выбрать, какой оригинальный портрет заменить
    (например, FD026S00).)

    Чтобы перезаписать исходный файл .DBI, добавьте флаг "ow":

            python fd_portrait_codec_v2_36.py replace CFDW0048.png any.DBI FD026S00 ow

После кодирования скрипт создаст файл any_mod.DBI, который можно
скопировать в папку Imgs1, Imgs2 или Interf игры Disciples Sacred Lands,
переименовать в any.DBI и заменить оригинальный файл (предварительно
обязательно сделайте резервную копию).
